# 实施计划：Agent-服务器通信加密（第二阶段收尾）

## 概述

第一阶段（批量传输、GZIP压缩、异步处理、重试机制）和第二阶段核心加密组件已完成实现。本任务列表聚焦于尚未完成的工作：密钥轮换通知与同步、服务器公钥分发完善、属性测试（PBT）、性能测试和生产部署配置。

- [x] 2. 完善密钥轮换通知与同步机制
  - [x] 2.1 在 `AgentEncryptionContext` 中实现密钥轮换触发逻辑

    - _需求：10.2、10.3_
  - [x] 2.3 实现 Agent 收到 HTTP 403 时的密钥重新注册逻辑
    - 在 `EncryptedBatchLogCollector` 的错误处理中，403 响应触发重新生成密钥对并注册
    - 注册成功后重试当前批次
    - _需求：12.3_
  - [ ]* 2.4 为密钥轮换流程编写单元测试
