脚本安装 LightScript 客户端

使用一键安装脚本，快速部署 LightScript 客户端到您的服务器

🐧 Linux
🍎 macOS
🪟 Windows

1 一键安装脚本

复制并运行以下命令，自动下载并安装 LightScript 客户端：

curl -fsSL http://8.138.114.34/scripts/install-linux.sh | bash -s -- --server=http://8.138.114.34:8080

2 手动安装步骤

如果一键安装失败，可以按照以下步骤手动安装：

下载客户端

wget http://8.138.114.34/releases/lightscript-agent-linux.tar.gz
tar -xzf lightscript-agent-linux.tar.gz
cd lightscript-agent

配置服务

# 编辑配置文件
sudo nano agent.properties

# 设置服务器地址
server.url=http://8.138.114.34:8080
register.token=your-register-token

启动服务

# 安装为系统服务
sudo ./install-service.sh

# 启动服务
sudo systemctl start lightscript-agent
sudo systemctl enable lightscript-agent

# 查看状态
sudo systemctl status lightscript-agent
⚠️ 注意事项
• 需要 root 权限安装系统服务
• 确保服务器能访问 8.138.114.34:8080
• 建议在防火墙中开放必要端口

1 一键安装脚本

复制并运行以下命令，自动下载并安装 LightScript 客户端：

curl -fsSL http://8.138.114.34/scripts/install-macos.sh | bash -s -- --server=http://8.138.114.34:8080

2 Homebrew 安装

如果您使用 Homebrew，可以通过以下方式安装：

# 添加 tap
brew tap lightscript/tap

# 安装客户端
brew install lightscript-agent

# 启动服务
brew services start lightscript-agent
⚠️ 注意事项
• macOS 可能需要在"安全性与隐私"中允许运行
• 建议使用 Homebrew 进行安装和管理
• 首次运行可能需要管理员权限

1 PowerShell 安装脚本

以管理员身份运行 PowerShell，执行以下命令：

Set-ExecutionPolicy Bypass -Scope Process -Force
iex ((New-Object System.Net.WebClient).DownloadString('http://8.138.114.34/scripts/install-windows.ps1'))

2 手动安装步骤

如果脚本安装失败，可以手动下载并安装：

下载并解压

# 下载安装包
Invoke-WebRequest -Uri "http://8.138.114.34/releases/lightscript-agent-windows.zip" -OutFile "lightscript-agent.zip"

# 解压到程序目录
Expand-Archive -Path "lightscript-agent.zip" -DestinationPath "C:\Program Files\LightScript"

安装为 Windows 服务

# 进入安装目录
cd "C:\Program Files\LightScript"

# 安装服务
.\install-service.bat

# 启动服务
net start LightScriptAgent
⚠️ 注意事项
• 需要以管理员身份运行 PowerShell
• 可能需要修改执行策略允许脚本运行
• 确保 Windows Defender 不会阻止安装

验证安装

安装完成后，请验证客户端是否正常运行

检查客户端状态

命令行检查

# Linux/macOS
sudo systemctl status lightscript-agent

# Windows
sc query LightScriptAgent
