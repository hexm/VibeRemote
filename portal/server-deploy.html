1 Docker 部署

系统要求

  • Docker 20.10+ 和 Docker Compose 2.0+
  • 2GB+ 可用内存
  • 10GB+ 可用磁盘空间
  • 开放端口：8080 (Web服务), 3000 (前端)

快速启动

# 下载 docker-compose.yml
curl -O https://raw.githubusercontent.com/lightscript/lightscript/main/docker-compose.yml

# 启动服务
docker-compose up -d

# 查看状态
docker-compose ps

完整部署步骤

1. 创建部署目录

mkdir lightscript-server
cd lightscript-server

2. 创建 docker-compose.yml

version: '3.8'

services:
  lightscript-server:
    image: lightscript/server:latest
    container_name: lightscript-server
    ports:
      - "8080:8080"
    environment:
      - SPRING_PROFILES_ACTIVE=docker
      - DATABASE_URL=jdbc:h2:file:/data/lightscript
    volumes:
      - ./data:/data
      - ./logs:/app/logs
    restart: unless-stopped
    depends_on:
      - lightscript-web

  lightscript-web:
    image: lightscript/web:latest
    container_name: lightscript-web
    ports:
      - "3000:80"
    environment:
      - REACT_APP_API_URL=http://localhost:8080
    restart: unless-stopped

volumes:
  data:
  logs:

3. 启动服务

# 启动所有服务
docker-compose up -d

# 查看日志
docker-compose logs -f

# 停止服务
docker-compose down
⚠️ 注意事项
• 首次启动可能需要几分钟下载镜像
• 确保端口 8080 和 3000 未被占用
• 数据将保存在 ./data 目录中

2 手动部署

系统要求

  • Java 8+ (推荐 Java 11)
  • Node.js 16+ 和 npm/yarn
  • 2GB+ 可用内存
  • 10GB+ 可用磁盘空间

部署步骤

1. 下载服务端

# 下载服务端
wget https://github.com/lightscript/lightscript/releases/latest/download/lightscript-server.jar

# 下载前端
wget https://github.com/lightscript/lightscript/releases/latest/download/lightscript-web.tar.gz
tar -xzf lightscript-web.tar.gz

2. 配置服务端

# 创建配置文件 application.yml
cat > application.yml << EOF
server:
  port: 8080

spring:
  datasource:
    url: jdbc:h2:file:./data/lightscript
    driver-class-name: org.h2.Driver
    username: sa
    password: 

  jpa:
    hibernate:
      ddl-auto: update
    show-sql: false

logging:
  level:
    com.example.lightscript: INFO
  file:
    name: logs/lightscript-server.log
EOF

3. 启动服务端

# 创建必要目录
mkdir -p data logs

# 启动服务端
java -jar lightscript-server.jar --spring.config.location=application.yml

# 后台运行
nohup java -jar lightscript-server.jar --spring.config.location=application.yml > logs/server.log 2>&1 &

4. 部署前端

# 安装依赖
cd lightscript-web
npm install

# 构建生产版本
npm run build

# 使用 nginx 或其他 web 服务器部署 build 目录
# 或者使用 serve 快速启动
npx serve -s build -l 3000
⚠️ 生产环境建议
• 使用 systemd 或其他进程管理器管理服务
• 配置 nginx 反向代理
• 使用外部数据库（MySQL/PostgreSQL）
• 配置 SSL 证书
