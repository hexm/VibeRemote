# 📦 下载前端库文件说明

所有HTML文件已配置为使用本地库文件，请按以下步骤下载库文件。

## 🚀 快速开始（3种方法）

### 方法1：使用Python脚本（推荐） ⭐

```bash
# 在web目录下执行
python download-libs.py
```

### 方法2：使用浏览器批量下载

打开浏览器，依次访问以下链接，右键 → 另存为：

1. **Vue.js** (150KB)
   - https://cdn.jsdelivr.net/npm/vue@3/dist/vue.global.prod.js
   - 保存为：`web/lib/vue/vue.global.prod.js`

2. **Element Plus JS** (2MB)
   - https://cdn.jsdelivr.net/npm/element-plus@2/dist/index.full.min.js
   - 保存为：`web/lib/element-plus/index.full.min.js`

3. **Element Plus CSS** (250KB)
   - https://cdn.jsdelivr.net/npm/element-plus@2/dist/index.min.css
   - 保存为：`web/lib/element-plus/index.min.css`

4. **Axios** (15KB)
   - https://cdn.jsdelivr.net/npm/axios@1/dist/axios.min.js
   - 保存为：`web/lib/axios/axios.min.js`

5. **ECharts** (900KB)
   - https://cdn.jsdelivr.net/npm/echarts@5/dist/echarts.min.js
   - 保存为：`web/lib/echarts/echarts.min.js`

### 方法3：使用wget或curl命令

```bash
cd web

# 创建目录
mkdir -p lib/vue lib/element-plus lib/axios lib/echarts

# 下载Vue
curl -L -o lib/vue/vue.global.prod.js https://cdn.jsdelivr.net/npm/vue@3/dist/vue.global.prod.js

# 下载Element Plus
curl -L -o lib/element-plus/index.full.min.js https://cdn.jsdelivr.net/npm/element-plus@2/dist/index.full.min.js
curl -L -o lib/element-plus/index.min.css https://cdn.jsdelivr.net/npm/element-plus@2/dist/index.min.css

# 下载Axios
curl -L -o lib/axios/axios.min.js https://cdn.jsdelivr.net/npm/axios@1/dist/axios.min.js

# 下载ECharts
curl -L -o lib/echarts/echarts.min.js https://cdn.jsdelivr.net/npm/echarts@5/dist/echarts.min.js
```

## ✅ 验证下载

下载完成后，目录结构应该如下：

```
web/
└── lib/
    ├── vue/
    │   └── vue.global.prod.js      (约150KB)
    ├── element-plus/
    │   ├── index.full.min.js       (约2MB)
    │   └── index.min.css           (约250KB)
    ├── axios/
    │   └── axios.min.js            (约15KB)
    └── echarts/
        └── echarts.min.js          (约900KB)
```

## 🎯 完成后

1. 启动Web服务器：
   ```bash
   cd web
   python -m http.server 3000
   ```

2. 访问：http://localhost:3000

3. 页面应该快速加载，无需依赖外部CDN！

## ⚡ 性能提升

- **加载速度**：从CDN的3-5秒降至本地的0.5-1秒
- **稳定性**：不受CDN网络影响
- **离线可用**：完全离线环境也可运行

## ❓ 常见问题

**Q: 下载失败怎么办？**
A: 可以尝试使用国内镜像或直接从浏览器下载。

**Q: 文件大小总共多大？**
A: 约3.3MB（压缩后）

**Q: 可以使用npm安装吗？**
A: 可以，但需要从node_modules复制文件到lib目录。
