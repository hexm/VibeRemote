## 技术细节

### 修复代码
```javascript
// 监听编辑模式，确保权限正确加载
useEffect(() => {
  if (modalVisible && modalType === 'edit' && currentUser) {
    // 使用setTimeout确保Modal和Checkbox.Group完全渲染后再设置值
    const timer = setTimeout(() => {
      const userPermissions = currentUser.permissions || []
      setSelectedPermissions(userPermissions)
      form.setFieldsValue({
        email: currentUser.email,
        realName: currentUser.realName,
        permissions: userPermissions
      })
    }, 100)
    
    return () => clearTimeout(timer)
  }
}, [modalVisible, modalType, currentUser, form])
```

### 关键改进
1. **添加form到依赖数组**: 确保form对象变化时重新执行
2. **使用setTimeout延迟100ms**: 确保Modal和Checkbox.Group完全渲染
3. **清理定时器**: 避免内存泄漏
