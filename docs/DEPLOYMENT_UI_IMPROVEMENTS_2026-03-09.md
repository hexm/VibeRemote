**新增功能**: 为每个客户端添加"状态收集"按钮

**功能说明**:
- 在客户端列表的操作列添加绿色同步图标按钮
- 点击后自动创建状态收集任务
- 根据操作系统自动选择合适的脚本：

**收集的信息包括**:
- 系统信息（OS版本、架构）
- CPU使用率
- 内存使用情况
- 磁盘使用情况
- 网络接口信息
- 运行中的进程（Top 10）
