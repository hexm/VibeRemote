**日期**: 2026年3月14日

## 📋 功能概述

实现了完整的Agent客户端安装功能，包括管理后台的安装入口和门户网站的下载/脚本安装页面。

## 🎯 功能特性

### 1. 管理后台安装入口
- **位置**: 客户端管理页面，刷新按钮旁边
- **交互**: 下拉菜单形式，提供两个安装选项
- **选项**:
  - 下载安装 → 跳转到门户下载页面
  - 脚本安装 → 跳转到门户脚本安装页面

### 2. 门户下载页面 (`/download.html`)
- **功能**: 提供各操作系统的安装包下载
- **支持系统**: Windows、Linux、macOS
- **特性**:
  - 响应式卡片布局
  - 操作系统图标和特性说明
  - 安装步骤指导
  - 版本信息展示

### 3. 门户脚本安装页面 (`/install.html`)
- **功能**: 提供一键安装脚本
- **支持系统**: Linux、macOS、Windows
- **特性**:
  - 标签页切换不同系统
  - 代码块复制功能
  - 详细安装步骤
  - 验证安装指导

## 📋 安装脚本内容

### Linux一键安装
```bash
curl -fsSL http://8.138.114.34/scripts/install-linux.sh | bash -s -- --server=http://8.138.114.34:8080
```

### macOS一键安装
```bash
curl -fsSL http://8.138.114.34/scripts/install-macos.sh | bash -s -- --server=http://8.138.114.34:8080
```

### Windows PowerShell安装
```powershell
Set-ExecutionPolicy Bypass -Scope Process -Force
iex ((New-Object System.Net.WebClient).DownloadString('http://8.138.114.34/scripts/install-windows.ps1'))
```
