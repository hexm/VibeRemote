# 任务创建界面优化 - TAB设计

## 概述

根据用户反馈，将任务创建窗口重新设计为TAB形式，参考脚本管理页面的设计，使脚本执行和文件传输功能更加清晰分离。

## 改进内容

### 1. TAB设计
- **脚本执行TAB**: 用于创建脚本执行任务
- **文件传输TAB**: 用于创建文件传输任务
- 每个TAB都有独立的表单和配置项

### 2. 界面优化
- 移除了原来的任务类型下拉选择
- 使用TAB切换来区分任务类型
- 每个TAB有不同的默认任务名称前缀
- 按钮文本根据当前TAB动态变化

### 3. 用户体验改进
- 更直观的任务类型选择
- 减少了用户的选择步骤
- 界面更加清晰，功能分离明确
- 参考了脚本管理页面的成功设计模式

## 技术实现

### 前端变更
- 添加了 `activeTaskTab` 状态管理
- 引入了 `Tabs` 和 `TabPane` 组件
- 重构了创建任务模态框的结构
- 更新了任务创建逻辑

### 代码文件
- `web-modern/src/pages/Tasks.jsx`: 主要修改文件

### 新增状态
```javascript
const [activeTaskTab, setActiveTaskTab] = useState('script')
```

### TAB结构
```javascript
<Tabs activeKey={activeTaskTab} onChange={setActiveTaskTab}>
  <TabPane tab={<span><CodeOutlined />脚本执行</span>} key="script">
    // 脚本执行表单
  </TabPane>
  <TabPane tab={<span><CloudDownloadOutlined />文件传输</span>} key="file-transfer">
    // 文件传输表单
  </TabPane>
</Tabs>
```

## 功能特性

### 脚本执行TAB
- 任务名称（默认前缀：脚本任务_）
- 执行客户端选择（手动/分组）
- 脚本来源（自定义/已有脚本）
- 脚本类型和内容
- 执行参数（超时时间、启动选项）

### 文件传输TAB
- 任务名称（默认前缀：文件传输任务_）
- 执行客户端选择（手动/分组）
- 文件选择和目标路径
- 传输选项（超时、覆盖、校验）

## 用户界面

### 按钮动态文本
- 脚本执行TAB: "创建脚本任务"
- 文件传输TAB: "创建文件传输任务"

### 图标设计
- 脚本执行: `<CodeOutlined />` 
- 文件传输: `<CloudDownloadOutlined />`

## 兼容性

- 保持了原有的后端API接口不变
- 任务创建逻辑根据TAB状态自动判断任务类型
- 所有现有功能保持完整
