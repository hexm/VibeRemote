## 🛠️ 安装步骤

### 第一步: 安装 Node.js

**macOS (推荐使用 Homebrew):**
```bash
# 安装 Homebrew (如果未安装)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 安装 Node.js
brew install node

# 验证安装
node --version
npm --version
```

**或者下载官方安装包:**
1. 访问 https://nodejs.org/
2. 下载 LTS 版本 (推荐 18.x 或更高)
3. 运行安装包并按提示安装

### 第二步: 启动新版前端

```bash
# 进入项目目录
cd LightScript/web-modern

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 或使用启动脚本
./start-modern.sh
```

### 第三步: 访问新界面

- **新版前端**: http://localhost:3001
- **旧版前端**: http://localhost:3000 (如果还在运行)
- **后端API**: http://localhost:8080

## 🔄 迁移策略

### 方案一: 并行运行 (推荐)
- 保持旧版本运行在 3000 端口
- 新版本运行在 3001 端口
- 逐步测试和验证功能
- 确认无误后切换到新版本

### 方案二: 直接替换
- 停止旧版前端服务

- 将新版本部署到 3000 端口

## 🐛 故障排除

### 问题1: npm 命令不存在
```bash
# 检查 Node.js 安装
which node
node --version

# 重新安装 Node.js
brew install node
```

### 问题2: 端口被占用
```bash
# 查看端口占用
lsof -i :3001

# 杀死占用进程
kill -9 <PID>

# 或使用其他端口
npm run dev -- --port 3002
```

### 问题3: 依赖安装失败
```bash
# 清理缓存
npm cache clean --force
rm -rf node_modules package-lock.json

# 重新安装
npm install
```
