## API权限验证

### 用户管理API
- `GET /api/web/users` → 需要 `user:view`
- `GET /api/web/users/{id}` → 需要 `user:view`
- `POST /api/web/users` → 需要 `user:create`
- `PUT /api/web/users/{id}` → 需要 `user:edit`
- `DELETE /api/web/users/{id}` → 需要 `user:delete`
- `POST /api/web/users/{id}/reset-password` → 需要 `user:edit`
- `POST /api/web/users/{id}/toggle-status` → 需要 `user:edit`
